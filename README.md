# Gabriel Lindsay - Culinary Consulting Website

A professional, responsive website for Gabriel Lindsay's culinary consulting business.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Service Showcase**: Highlights 6 core consulting services
- **Contact Form**: Easy-to-use contact form for inquiries
- **Smooth Navigation**: Scroll animations and mobile-friendly menu

## Sections

1. **Home/Hero**: Eye-catching introduction with call-to-action
2. **About**: Background and expertise overview
3. **Services**: Six key consulting services with descriptions
4. **Contact**: Contact form for client inquiries

## How to Use

1. Open `index.html` in your web browser to view the site locally
2. Customize the content in `index.html` to match your specific experience and services
3. Adjust colors in `styles.css` (see CSS variables at the top)
4. Deploy to a hosting service (Netlify, Vercel, GitHub Pages, etc.)

## Deployment Options

### Netlify (Recommended)
1. Create account at netlify.com
2. Drag and drop the folder
3. Connect custom domain

### Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the project directory
3. Follow prompts

### GitHub Pages
1. Create GitHub repository
2. Push code to repository
3. Enable GitHub Pages in settings

## Customization Tips

- Update the hero section text to match your personal brand
- Add real testimonials if available
- Include portfolio images of your work
- Connect the contact form to a backend service (Formspree, EmailJS, etc.)
- Add your social media links in the footer
