# Dolphins to the Max

An infinitely zoomable dolphin scene, inspired by [Koalas to the Max](https://www.koalastothemax.com).
One illustration is drawn recursively into a 3x3 grid of itself, forever — zoom into any point
and the whole picture appears again. Scroll/pinch to zoom, drag to pan.

Pure HTML/CSS/canvas, no dependencies. `images/dolphin-scene.svg` is an original illustration.

Deployed via GitHub Pages (see `CNAME`).
